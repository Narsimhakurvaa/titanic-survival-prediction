##  Project Overview
The Titanic dataset contains demographic and travel details of passengers.  
The objective is to predict whether a passenger survived based on specific features.

### Tasks Completed
- Data loading and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature Engineering:
  - FamilySize
  - IsAlone
  - AgeBand
- Model training & comparison
- Performance evaluation

##  Models Compared
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

The model with the best performance was selected based on **accuracy score**.

---

##  Key Insights (EDA)
- Females had a significantly higher survival rate
- Survival probability was higher for 1st-class passengers
- Travelling alone reduced survival chance
- Younger passengers showed better survival trends
